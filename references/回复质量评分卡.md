# 回复质量评分卡

> 给嘴替产出的每一次三档回复打分——不是"这条发出去安不安全"（那是 [`风险评分卡.md`](风险评分卡.md) 管的），而是"这次的回复本身写得好不好"。即使风险判断对了，回复也可能写得平、不像人、三档分不清。
>
> 两张卡分工：**风险卡管该给多强，质量卡管写得好不好。两张都过，才算合格输出。**

## 五个维度

每维 0-2 分，满分 10。**三档识别度和边界安全是一票否决**——识别度不够，三档就失去存在意义；边界不过，回复就变成给用户挖坑。

| 维度 | 0 分 | 1 分 | 2 分 |
|---|---|---|---|
| **潜台词命中** | 只复述字面，或写成"对方存在边界感缺失，需有效沟通"这种正确的废话 | 说到了诉求，但没点破成本流向 | 一句戳中"对方想要什么 + 用户要注意什么"，点破成本转移 |
| **三档识别度** | 盖住标签认不出是哪档 | 体面/掀桌能分，但绵里藏针不像绵里藏针 | 盖住标签也一眼三档分明，各有招牌动作 |
| **可发送性** | 有脏字/攻击/"你可以这样说……"，发不出去 | 能发，但接不住后果（惹祸、留把柄） | 直接复制可发，且接得住后果 |
| **口吻贴合** | 一眼 AI 味，或渠道/关系错位（给伴侣写成职场谈判） | 大致像，但有出戏的词 | 匹配渠道和关系，像真人（专属嘴替则像用户本人） |
| **边界安全** | 触碰掀桌红线（脏字/人身攻击/威胁/把猜测当事实） | 擦边，需微调 | 旁人看了都觉得有理有节，还给了对方台阶 |

## 评级

| 总分 | 评级 | 处理 |
|---|---|---|
| 9-10 | A 级 | 放行 |
| 6-8 | B 级 | 挑扣分那一维重写 |
| ≤5 | C 级 | 推翻重写 |

任一"一票否决"维度得 0 分，无论总分多少，直接判 C 级重写。

## 自检流程

每次输出前，按这个顺序走一遍：

1. **潜台词命中**了吗？说不清就别急着写回复，回到[信号体系](潜台词判断框架.md)重判。
2. **三档识别度**够吗？做"盖标签盲读"（见下）——自己盖住标签读一遍，分不清就重写到能分清。
3. **可发送性**过吗？逐档默念"我现在把它复制发出去，会怎样"。发不出去的一律改。
4. **口吻贴合**吗？确认渠道（微信/邮件）和关系位置（上级/平级/家人）匹配。
5. **边界安全**过吗？尤其掀桌版，套用[风险评分卡](风险评分卡.md)的 5 个安全问题，任一不过就重写。

## 盖标签盲猜：识别度的可复现验证

"三档到底分不分得清"，任何人五分钟就能自己验证，这也是本 skill 最该被检验的能力：

1. 拿嘴替对一条消息生成的三档回复。
2. **抹掉【体面版】【绵里藏针版】【掀桌版】三个标签**，只留三段正文，打乱顺序。
3. 找一个没看过原文的第三方，让他只读这三段，盲猜哪段是哪档。
4. 猜对的比例，就是这次输出的**识别度**。

判定：三段都能被稳定猜对 → 识别度合格；有人把绵里藏针猜成体面、或把掀桌猜成绵里藏针 → 不合格，重写。

## 一个满分样本（旗舰案例"朋友借钱不还"）

对方："最近手头是不是宽裕点了？上次那个事儿……你懂的哈，不急不急。"

- **体面版**：`哈哈还那样～对了正好说到，上次那笔你看方便的时候安排一下就行，不着急，给我留个心就好。`
  → 招牌动作：把话头接住、气氛不尴尬，但一个字没接他"你宽裕吗"的套。
- **绵里藏针版**：`我这边紧紧巴巴的哈～说到"那个事儿"，我也一直记着呢，你看这个月方便不？不用一次性，分次也行，咱把它了了，彼此都轻松。`
  → 招牌动作：一根针"我这边紧紧巴巴的哈"（暗示"我也不宽裕"反打他的试探），藏在"给台阶、分次也行"的体贴里。
- **掀桌版**：`宽裕谈不上哈。既然你提到上次那事，咱就说清楚：那笔钱到现在三个月了，你看什么时候能还？定个时间我心里也有数。不是催你，是账还是得算明白，咱这朋友才处得长久。`
  → 招牌动作：句子明显更短更直，点破"不急"的拖字诀，把"三个月""定个时间"摆上桌，最后给一个"照规矩来朋友才处得长久"的台阶。

盖住三个标签给人读，绝大多数人能凭"一个字不接他的套 / 藏了一根针 / 短句点破拖字诀"把三档分出来——这就是识别度合格的样子。
