# 嘴替.skill

**你只管憋屈，回消息的事交给它。**

一个 Agent Skill。把任何难回、憋屈、不知道怎么开口的消息丢进来——它先翻译对方的潜台词，再给你三档可以直接发出去的回复，最后附一句长期策略。

老板画饼、甲方压价、亲戚催婚、伴侣阴阳怪气、前任深夜发消息、同事夹枪带棒——所有"这条我到底该怎么回"的时刻，你都有嘴替了。

基于开放的 Agent Skills 协议，可在 Claude Code、Codex、Cursor、OpenClaw 等兼容 runtime 中运行。

## 看效果

**朋友借钱不还那套**（同一条消息三种姿态，差别一眼可见）

> **朋友**（借钱三个月没还）❯ 最近手头是不是宽裕点了？上次那个事儿……你懂的哈，不急不急。
>
> **嘴替** ❯
> 🎭 他欠你钱，却用"你是不是宽裕了"把话题反转——试探你会不会主动提，或者暗示你"有钱就别催"。"不急不急"是话术：真不急的人，不会专门发这条。
>
> 【体面版】哈哈还那样～对了正好说到，上次那笔你看方便的时候安排一下就行，不着急，给我留个心就好。
>
> 【绵里藏针版】我这边紧紧巴巴的哈～说到"那个事儿"，我也一直记着呢，你看这个月方便不？不用一次性，分次也行，咱把它了了，彼此都轻松。
>
> 【掀桌版】宽裕谈不上哈。既然你提到上次那事，咱就说清楚：那笔钱到现在三个月了，你看什么时候能还？定个时间我心里也有数。不是催你，是账还是得算明白，咱这朋友才处得长久。
>
> 🧭 模糊的欠账最伤朋友。每次都平静地把"定个时间"摆出来——不吵、不翻旧账，把它变成有期限的事，反而是保护这段关系。

**老板周末派活那套**

> **老板**（周五 18:47）❯ 在吗？下周一早上我约了大老板过方案，PPT 周末你帮我优化优化。不用做太精细，你效率高，半天就搞定了。
>
> **嘴替** ❯
> 🎭 "不用太精细""半天就搞定"不是体谅你，是提前把这活定性成小事——让你既不好意思拒绝，也不好意思提补偿。
>
> 【绵里藏针版】好的领导。不过"半天搞定"我不敢打包票——给大老板看的东西，我不想按"不用太精细"的标准来。我周末排一下，周日晚上给您终版；这周末的时间我先记成调休哈，下周找天补上。
>
> 🧭 活越被说得轻，越不用给你补偿。以后凡是"很简单/顺手/半天"的活，都自然地报一个真实工作量再接。

**甲方改回第一版那套**

> **甲方**（第五轮修改后）❯ 我们领导看了，还是觉得第一版顺眼，改回第一版吧。尽快哈，上线时间不变。
>
> **嘴替** ❯
> 🎭 折腾五版又回到起点，说明他们内部没人能拍板，试错成本全让你垫。"上线时间不变"最扎心——他们决策反复浪费的时间，默认从你的工期里扣。
>
> 【掀桌版】可以改回第一版，但两件事说在前面：一，前后五轮修改都是按你们的意见走的，工时是实际发生的成本，我整理个明细，咱们聊聊怎么算；二，"上线时间不变"的前提是这次锁版——领导书面确认第一版之后不再改。两个都定了，我今天就动工。

**妈妈催婚那套**

> **妈妈** ❯ 王阿姨给你介绍了个男孩子，条件真的很好，见一面怎么了？妈不图你什么，就这一个心事。你别嫌我们烦，我跟你爸年纪都上来了。
>
> **嘴替** ❯
> 🎭 先说清楚：这条没有套路，焦虑是真的，爱也是真的。她要的不是你立刻结婚，是"这事有人在管"的确定感——别当成攻击去防御，她越被顶回去越焦虑。
>
> 【绵里藏针版】妈，人我可以见。但有件事想跟你说清楚：我不是不上心，是想找对的，不想凑合。你和爸的心事我懂，但这个心事别背在你们身上——我日子过得挺好，这本身就该让你们放心才对。

## 安装

打开你在用的 agent（Claude Code / Codex / Cursor / OpenClaw 等），告诉它：

```
帮我安装这个 skill：https://github.com/SilentFleetKK/zuiti-skill
```

或用通用 CLI 安装器：

```
npx skills add SilentFleetKK/zuiti-skill
```

它会自动识别你的 runtime 并放到正确目录。想指定环境就加 `-a claude-code` / `-a codex` / `-a cursor` 等参数。

即使你的 runtime 不支持自动加载，也可以直接把 `SKILL.md` 的内容粘进对话——它本质就是一份 markdown。

手动安装路径见 [INSTALL.md](INSTALL.md)。

## 怎么用

装好后，遇到难回的消息，直接把它贴给你的 agent：

```
> 帮我回一下这条：[粘贴对方发来的消息]
> 这话什么意思，我该怎么回：[消息]
> 嘴替，[消息]
```

你会拿到固定的四段：

1. **🎭 潜台词翻译** —— 对方真正想干什么，一针见血
2. **💬 三档回复** —— 体面版 / 绵里藏针版 / 掀桌版，每档都能直接复制发送
3. **🧭 长期策略** —— 这类消息往后怎么处理更省心

挑一档，复制，发出去。完事。

## 你的专属嘴替

默认的嘴替说话是"标准得体"的。但你可以让它**学你本人的口吻**，之后所有回复读起来就像你自己打的字，而不是一看就是代笔。

```
> 学我说话，这是我平时的聊天记录：[粘贴/上传]
> 蒸馏我的风格，建一个我的专属嘴替
```

它会提炼你的口头禅、句子长短、标点和表情习惯、情绪底色，生成一个专属 skill 存到 `zuiti-personas/`，触发词你自己起（比如 `/我的嘴替`）。

用得越久越像你。哪句不像，说一声"我不会这么说"，它就改。

## 它守住的分寸

嘴替的"最狠档位"是把道理讲透、守住底线、不撕破脸——不带脏字、不搞人身攻击、不教唆报复。因为真正解气的从来是"体面地赢"，不是"一起下头"。发出去会让你自己难堪或惹麻烦的话，不算好嘴替。

它给你三档，你自己挑。它不替你决定该忍还是该撕——选择权永远在你手里。
